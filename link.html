Link:

definition:

  1. The link tag in HTML, also known as the anchor tag, is used to create a hyperlink that can be clicked to navigate to another web page or a specific location within the same page.

  2. It is defined using the <a> and </a> tags and includes the destination URL or the location within the same page as the value of the href attribute.

  3. The text or image that appears as a hyperlink is placed between the opening and closing tags. The link tag can also be used to link to email addresses, files, and other resources.

  4. It can be used for Navigations as well.

  6. or you can put relative link (link present in you folder) like LISTS.
